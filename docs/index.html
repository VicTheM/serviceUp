serviceUp

Hire like a pro!

serviceUp is a website where you can hire professionals in your locality for all sort of craftsmanship and handwork.

Features

  • Easy-to-use interface: Just one search and you are linked with the professional
  • Optimized for tight time-constraints
  • Transparency: You get to view the professional profile and see his past work
  • The highest rated and closest to you will always be assigned to you on every search

About

This site was built as a Capstone project for ALX Software Engineering course. It was inspired by my lack of getting clients consistently (I am an Electrician by profession) and the need to have a platform where I can be easily found by clients. I decided to build a platform where professionals can be easily found by clients.
I sat and saw other electrician coe from a very far place to work for people in my area, and I wondered, "How can they bypass me whe I am closer?" so proximity, authenticity and availability are the three factors that determine which professional is assigned to a client.

How it was built

Amazingly, it was built in 3 weeks. I used Flask for the backend, and pure HTML, CSS and JS for the frontend. This current landing page is being hosted on giuthub pages, and the button above that says "Goto site" will take you to the actual site. The main backend is hosted on an ubuntu machine, and served via gunicorn which gets proxy request from an Nginx web server.
